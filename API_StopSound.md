# StopSound

Use StopSound() to stop any sound playing on a specific channel.

## Usage

`StopSound ( channel )`

## Arguments

<table>
  <tr>
    <td>Name</td>
    <td>Value</td>
    <td>Description</td>
  </tr>
  <tr>
    <td>channel</td>
    <td>int</td>
    <td>The channel ID to stop a sound on.</td>
  </tr>
</table>
